# 部署指南

## 构建生产版本

在部署之前，需要先构建生产版本：

```bash
npm install
npm run build
```

构建完成后，所有文件会生成在 `dist/` 目录中。

## 腾讯云静态网站托管部署步骤

### 1. 准备工作

- 确保已注册腾讯云账号
- 完成实名认证
- 准备域名（可选，也可以使用COS提供的默认域名）

### 2. 创建COS存储桶

1. 登录腾讯云控制台：https://console.cloud.tencent.com/
2. 进入"对象存储 COS"服务
3. 点击"创建存储桶"
4. 配置存储桶：
   - **名称**：自定义（如：delta-menus）
   - **所属地域**：选择离用户最近的地域（如：北京、上海、广州）
   - **访问权限**：选择"公有读私有写"或"公有读写"
   - **存储类型**：标准存储
5. 点击"创建"

### 3. 开启静态网站托管

1. 进入创建的存储桶
2. 点击左侧菜单"基础配置"
3. 找到"静态网站"选项，点击"编辑"
4. 开启静态网站功能
5. 配置：
   - **索引文档**：`index.html`
   - **错误文档**：`index.html`（可选，用于SPA路由）
6. 保存配置

### 4. 上传文件

#### 方式1：通过控制台上传

1. 进入存储桶的"文件列表"
2. 点击"上传文件"
3. 选择 `dist/` 目录下的所有文件
4. 上传到存储桶根目录
5. 确保文件结构正确：
   ```
   存储桶根目录/
   ├── index.html
   ├── assets/
   │   ├── index-xxx.js
   │   └── index-xxx.css
   └── images/
       └── ...
   ```

#### 方式2：使用COS CLI工具（推荐批量上传）

1. 安装COS CLI：
   ```bash
   pip install coscmd
   ```

2. 配置COS CLI：
   ```bash
   coscmd config -a <SecretId> -s <SecretKey> -b <BucketName> -r <Region>
   ```

3. 上传文件：
   ```bash
   cd dist
   coscmd upload -r . /
   ```

#### 方式3：使用GitHub Actions自动部署（高级）

可以配置GitHub Actions，在代码推送时自动构建并部署到COS。

### 5. 配置自定义域名

1. 在存储桶的"域名管理"中，点击"自定义域名"
2. 添加您的域名（如：www.delta-menus.com）
3. 按照提示配置CNAME记录：
   - 登录您的域名DNS管理后台
   - 添加CNAME记录：
     - **主机记录**：www（或@）
     - **记录值**：COS提供的CNAME地址
     - **TTL**：600
4. 等待DNS解析生效（通常几分钟到几小时）

### 6. 配置HTTPS（推荐）

1. 在"自定义域名"配置中，开启HTTPS
2. 腾讯云会自动申请免费SSL证书
3. 等待证书颁发（通常几分钟）
4. 配置完成后，网站即可通过HTTPS访问

### 7. 配置CDN加速（可选但强烈推荐）

CDN可以大幅提升访问速度，特别是对于图片等静态资源。

1. 进入"内容分发网络 CDN"服务
2. 点击"添加域名"
3. 配置：
   - **加速域名**：您的域名（如：www.delta-menus.com）
   - **加速区域**：中国境内（或全球）
   - **源站类型**：COS源
   - **源站地址**：选择您的COS存储桶
4. 完成配置后，更新DNS解析：
   - 将域名的CNAME记录指向CDN提供的地址
5. 配置缓存规则（可选）：
   - HTML文件：不缓存或短时间缓存
   - JS/CSS文件：长时间缓存
   - 图片文件：长时间缓存

## 域名申请指南

### 腾讯云域名注册

1. 访问：https://dnspod.cloud.tencent.com/
2. 搜索您想要的域名
3. 选择合适的后缀（.com、.cn等）
4. 添加到购物车并完成支付
5. 完成域名实名认证

### 域名选择建议

- **长度**：建议6-15个字符
- **后缀**：
  - `.com`：国际通用，价格较高
  - `.cn`：国内域名，需要备案，价格较低
  - `.com.cn`：国内企业常用
- **关键词**：考虑与"三角洲行动"、"陪玩"相关的关键词

### 域名备案（如需要）

如果使用.cn域名或国内服务器，需要进行ICP备案：

1. **准备材料**：
   - 身份证正反面照片
   - 域名证书
   - 服务器信息

2. **备案流程**：
   - 登录腾讯云备案系统
   - 填写备案信息
   - 上传材料
   - 等待审核（7-20个工作日）
   - 备案通过后，域名即可正常使用

## 访问测试

部署完成后，访问您的域名，检查：

- [ ] 首页正常加载
- [ ] 商品列表正常显示
- [ ] 分类切换功能正常
- [ ] 商品详情页正常显示
- [ ] 图片正常加载
- [ ] 移动端显示正常
- [ ] HTTPS正常工作（如已配置）

## 常见问题

### 1. 页面显示404

- 检查是否开启了静态网站托管
- 检查索引文档是否配置为 `index.html`
- 检查文件是否正确上传到根目录

### 2. 路由跳转404（SPA路由问题）

- 确保错误文档也配置为 `index.html`
- 或者在COS中配置重定向规则

### 3. 图片无法加载

- 检查图片路径是否正确
- 检查COS访问权限是否为"公有读"
- 检查CDN缓存规则（如使用CDN）

### 4. HTTPS证书问题

- 等待证书颁发完成
- 检查域名DNS解析是否正确
- 确认域名已通过实名认证

## 维护和更新

更新网站内容：

1. 修改代码和数据文件
2. 重新构建：`npm run build`
3. 删除COS中的旧文件
4. 上传新的 `dist/` 目录文件

或者使用COS CLI：

```bash
cd dist
coscmd delete -r /
coscmd upload -r . /
```

## 成本估算

- **COS存储**：约0.1元/GB/月（按实际使用量）
- **CDN流量**：约0.15元/GB（按实际使用量）
- **域名**：约50-200元/年（一次性）
- **SSL证书**：免费（腾讯云提供）

对于小型网站，月成本通常在10-50元之间。
